## analisis_northwind
Dashboard interactivo de análisis de ventas con la base de datos de northwind (1996-1998), como proyecto de práctica 
para portafolio de análisis de datos.

## Herramientas utilizadas
- Power BI = visualización e informe
- DAX = medidas y cálculos
- SQL = Exploracion de datos (conocimientos de SQL)

## Contenido del repositorio
- 'Informe_northwnd.pdf' = reporte exportado de Power BI
- 'medidas_dax_northwind.md' = medidas dax utilizadas en el reporte
- 'Proyecto.sql' = Queries utilizados en proyecto personal

## Insights principals
- Margaret fue la vendedora top con $232,891
- Beverage fue la categoría más vendida con $267,871
- USA y Alemania lideraron las ventas geográficas

## NOTA
  - Los datos son históricos de práctica (Northwind) por lo que no se incluyeron metas reales.
